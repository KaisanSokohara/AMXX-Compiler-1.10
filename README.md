# Counter Strike 1.6 AMX Compiler (AMXMODX 1.10.x)
- I added as much as i can of Include's to make it easier to use.

# Usage
- Attach the Amxmodx Studio to "amxxpc.exe" via Tools > Settings > Compiler > Pawn-Compiler.
- Grab your sma file to the "compiler.exe" or "amxxpc.exe" and get it from compiled folder.

# Requirement's
- AMXMODX [ 1.10.x ]

# Information
- This might make some plugin's doesn't work on other AMXMODX Version's so be aware of that.
